# web-resume

A professional web-based resume portfolio project.

## Project Overview

This repository contains the source code for a modern, interactive web resume showcasing professional experience, skills, and projects.

## Getting Started

### Prerequisites
- Node.js (v14+)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone git@github.com:kaleblucas/web-resume.git

# Install dependencies
npm install

# Start development server
npm start
```
